# School Readiness Prediction System
### Data Attribute
- Age
- Gender
- Child Position
- Number of Siblings
- Have Pre-Elementary
- Father Education
- Mother Education
- School Readiness (Label)
### Class Label
- Ready (Disarankan)
- Doubt (Dipertimbangkan)
- Not Ready (Tidak Disarankan)
### Algorithm
- Artificial Neural Network
- Adam + Nesterov Momentum
